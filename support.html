Getting Started

Installation Guide

  1. Download the EA installer from licensedownload.veroxys.com
  2. Run the installer and follow the setup wizard
  3. Enter your unique License Key when prompted
  4. Configure your trading parameters
  5. Start with demo trading to verify settings

System Requirements

  • Windows 10/11 or macOS 10.14+
  • MetaTrader 5 platform
  • Stable internet connection (latency < 100ms recommended)
  • Minimum 4GB RAM, 8GB recommended
  • VPS hosting recommended for 24/7 operation

Frequently Asked Questions

License & Account Management

Q: How many accounts can I use with one license?

A: Each license allows installation on up to 3 trading accounts that you own or control.

Q: Can I transfer my license to a different account?

A: Yes, contact support to deactivate the old account and activate a new one.

Q: What happens if I exceed my account limit?

A: The EA will stop functioning. Contact support to purchase additional licenses.

Trading Configuration

Q: What are the recommended settings for each EA?

A: Default settings are optimized for each EA type:

  • Conservative: 0.01 lot size per $1,000 balance
  • Balanced: 0.02 lot size per $1,000 balance
  • Ultra-Aggressive: 0.05 lot size per $1,000 balance

Q: Can I modify the risk parameters?

A: Yes, but we recommend starting with default settings and adjusting gradually.

Q: What timeframes does the EA trade on?

A: Our EAs are optimized for M15 and H1 timeframes on the Micro E-Mini Nasdaq 100 (MNQ).

Technical Issues

Q: The EA is not placing trades. What should I check?

  • Verify AutoTrading is enabled in MT5
  • Check that markets are open
  • Ensure sufficient margin is available
  • Confirm EA is attached to the correct chart
  • Review the Experts tab for error messages

Q: I'm getting a "License Invalid" error.

A: This typically means:

  • License Key was entered incorrectly
  • Account number doesn't match registered accounts
  • Subscription payment failed (for monthly plans)

Troubleshooting Guide

Common Issues and Solutions

Connection Problems

  • Issue: "No connection to trade server"
  • Solution: Check internet connection, verify broker server settings, restart MT5

Performance Issues

  • Issue: EA running slowly or freezing
  • Solution: Close unnecessary charts, increase available RAM, consider VPS hosting

Trading Errors

  • Issue: "Not enough money" error
  • Solution: Reduce lot size, add funds, or check margin requirements

Best Practices

  1. Always test on demo first - Run for at least 2 weeks before live trading
  2. Monitor regularly - Check positions at least daily
  3. Keep software updated - Install updates within 48 hours of release
  4. Maintain trading journal - Document settings and performance
  5. Use VPS for reliability - Ensures 24/7 operation without interruption
